8.2.3 テキストデータのクレンジング

教科書のコードだと顔文字がくっついてしまうので、顔文字のjoinはwhitespaceを使うようにした。

def preprocessor(text):
    text = re.sub("<[^>]*>", "", text)
    emoticons = re.findall("(?::|;|=)(?:-)?(?:\)|\(|D|P)", text)
    text = re.sub("[\W]+", " ", text.lower()) + " ".join(emoticons).replace("-", "")
    return text
