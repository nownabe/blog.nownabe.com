SQLite3のモジュールがないとだめらしい。
sqliteをいれて、Pythonをインストールし直す。
pyenvを使っているのでこんな感じ。


$ pyenv uninstall 2.7.13
$ sudo dnf install -y sqlite-devel
$ pyenv install 2.7.13
