# Tech

* GAE/Goのアプリをいくつかデプロイして、次のようなことがわかってきた

    * ベンダリングはどうすればいいか

    * これらをMakefileにまとめる方法
    * GCPのAPIを使う方法
* Cloud Dataflowのレベルが上がった
    * テンプレートの使い方
    * 様々なLimitation
    * ValueProviderの使い方
    * APIの使い方
* Cloud Functionを本番導入した

* AWSのレベルアップ

    * CloudWatch + Lambda
    * Batch
* DataflowでサーバサイドKotlinを使い始めた
    * Kotlinこそは真のBetter Javaって感じがする

    * versioncakeとともに
